# Wyre's YES Compliance Token

See ![YESToken](YESToken.md) for the specification this is implementing.

### Getting Started

0. Install NPM
1. Install Truffle and darq-truffle as global dev deps:

 
       $ npm i -D -g truffle
        
    
2. npm install 

### Deploy

    $ truffle migrate [--reset]

### Run Tests

    $ truffle test
